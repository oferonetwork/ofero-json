Available Endpoints

GET

Verify Domain License

/api/v1/registry/verify/{domain}

Check if a domain is registered and has a valid license in the Ofero Network. Returns license tier, expiration, and organization details.

POST

Register Domain

/api/v1/registry/register

Register a new domain in the registry. Requires admin authentication. Set license tier, expiration, and organization information.

GET

List All Domains

/api/v1/registry/list

Retrieve a complete list of all registered domains. Requires admin authentication. Returns domain count and full registry.

GET

Health Check

/api/v1/registry/health

Check API status and availability. Returns service version, status, and current timestamp. No authentication required.

Quick Start Example

# Verify a domain license curl https://licence.ofero.network/api/v1/registry/verify/irigarden.ro # Response: { "valid": true, "domain": "irigarden.ro", "license": { "tier": "enterprise", "lifetime": true }, "organization": { "name": "Iri Garden" } }
